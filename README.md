# sangjjangit.github.io

jekyll 정적 웹 생성기를 통한 반응형 웹 블로그 생성.

1. jekyll 에 사용된 언어 및 템블릿
   - ruby, liquid, html, markdown 으로 파악됨.

2. 진행상황
   - jekyll 명령어로 기본 구성
   - 기본 뼈대 html, css, js 구성
   - 블로그 방문자수 hits 적용
     - hits 방문자수 확인은 페이지 로딩만 되어도 카운터가 올라가는 거라, 구글 애널리틱스을 통해서만 정확한 통계를 파악가능함.
     - 구글 애널리틱스에 사이트 등록.
       
   - 코드블럭 하이라이트 js 추가
   - 게시물 검색기능 추가
   - material css 적용
   - 2023.11.19 1차 마무리
   - 이제 블로그 포스팅 내용에 집중할 것!

3. 번외
   - 구글 애널리틱스 사이트 등록
   - 구글 애드센스 인증
     - 구글 광고 적용을 위한 사이트 인증, 소요시간 1주정도 예상
   - 구글 서치 콘솔 등록
     - 구글 검색에 노출이 되도록 설정
     - 차후 네이버, 다음도 등록 예정
     - jekyll plugins : sitemap, feed
